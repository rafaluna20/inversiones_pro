## 📈 Métricas Financieras Avanzadas

### 1. **Sharpe Ratio** (Retorno ajustado por riesgo)
```typescript
const sharpeRatio = (roiPromedio - tasaLibreRiesgo) / desviacionEstandarROI;
// Ejemplo: (45.3% - 5%) / 8.2% = 4.91 (Excelente)
```

### 2. **IRR (Internal Rate of Return)**
```typescript
// Tasa de retorno que hace el VPN = 0
const irr = calcularIRR(flujosDeCaja);
// Ejemplo: 52.3% anualizado
```

### 3. **CAGR (Compound Annual Growth Rate)**
```typescript
const cagr = Math.pow(valorFinal / valorInicial, 1 / años) - 1;
// Ejemplo: 38.7% anual compuesto
```

### 4. **Max Drawdown** (Pérdida máxima desde pico)
```typescript
const maxDrawdown = (picoHistorico - valleMinimo) / picoHistorico;
// Ejemplo: -5.2% (Muy bajo, excelente)
```
